# 图形比例分割可视化工具

## 图形文件上传

- 点击上传区域并选择本地的svg文件上传
- 本工具仅支持svg文件（svg是主要由节点定义的矢量图形，因为主要由节点定义，所以可以进行切分）
- 推荐两个图标下载网站，svg除了可以用在本工具之外，由于svg简约且可自定义颜色，也可以用于ppt制作
  - [Google Fonts](https://fonts.google.com/icons)图表数量多且可商用
  - 字节跳动旗下的图标库[IconPark](https://iconpark.oceanengine.com/home)，图标量较少但全部可以商用
  - [阿里巴巴矢量图标库](https://www.iconfont.cn/)，但使用时需注意甄别使用条件，部分可能无法用于商业用途

## 分割设置

- 可以通过滑动slider或者改写右侧的数字调整分块的百分比
  - 请注意，这里的百分比是当前的第x部分占总体的百分比
  - 请注意：百分比所有部分的百分比数值加和应等于100%
- 可以在下方的`添加分割点`按钮添加新的部分，或者在右侧删除按钮删去当前部分
- 每一部分可以自定义名称
- 每一部分有两个颜色选择框，左侧的颜色选择框是填充色，右侧的颜色选择框是描边色

## 标签大小设置

- 可以用slider滑动调节占比百分比&每一部分名称的文字标注的大小
- 可以用slider滑动调节文字离中心图案的距离，slider越靠右则图中的标注文字离中心越近
- `隐藏标注`按钮可以隐去所有图中的文字标注

## 缩放设置

- 用于调整svg图像的横纵比，垂直缩放的slider是用于调整svg图像在竖直方向上的比例，默认为原长，比例低于100%则在竖直方向上比原长更加短小

## 右侧展示栏

- 生成预览：生成分割后的svg图像预览
- 导出PNG/JPG：导出svg图像
